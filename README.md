## Installation
Download the repo. It will install in `~/bin/keylogger`.

Install it:

`$ git clone https://github.com/unamatasanatarai/keylogger-mac && cd keylogger-mac`

`$ make && make install`

Run it:

`$ keylogger .`

### Uninstall
`$ sudo make uninstall`
Will uninstall the program, but not the logs.
